<https://github.com/vuejs/vue-cli> 

### 安装

```bash
#查看vue-cli版本
$ npm show vue-cli
#安装版本确认
$ [sudo] npm install -g vue-cli@2.9.3
#查看安装的版本
$ vue -V
#命令帮助
$ vue -h
```



### 项目初始化

```bash
# 使用webpack模板初识化项目（项目名不能有大写）
$ vue init webpack myweb
$ cd myweb
# 运行项目
$ npm run dev
# 或者
$ npm start 
# 编译打包
$ npm run build
```
